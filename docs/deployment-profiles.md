# NEXUS CORE Deployment Profiles

## Purpose

The deployment profiles provide a staged adoption model for SMB environments.

## Profile Model

- Essential: baseline secure stack
- Essential Plus: stronger identity and access layer
- Intermediate: broader service coverage
- Intermediate Plus: automation and collaboration expansion
- Advanced: monitoring, evidence, and deeper operational intelligence

## Public Framing

These profiles are intended as product and planning categories rather than exact implementation templates.
